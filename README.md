# WEDE6021_DominantDuo_POEFINAL

Innovative Features added 
Item authentication
There is an authentication of products from sellers, admin are able to authenticate items, before publishing it for sale, the user will be contacted via email if its approved and then the admin can publish it.

Whatsapp Contacting
Users are able to contact the business via Whatsapp, by sending an image of their item to sell, there is a provided link with a dummy contact number.

Item Condition
admin is able to say what the condition of the item is and a bar on the item will show the live condition of the item to the users.

Another innovative feature was the ability to sell products by it was required as well by the assignment. Selling is available at the sell link
